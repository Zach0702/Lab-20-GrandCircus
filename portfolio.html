Lab 17 Grand Circus Prime Numbers Application

  • The above project allows the user to find out prime numbers
  • The user is prompted by asking which prime number they would like to find (ex: entering 1 would show you 2 as the first prime number and so on)
  • The user is allowed to complete this task as many times as they would like, and able to find any prime number that they would like
  • At the end the user is shown a summary of their session with the application
To check out my prime number application click the following link: Prime Numbers App

Rock Paper Scissors Application

  • The above project allows the user to play rock paper scissors against two types of computer players in the application
  • The first player the user can play against only selects rock everytime
  • The second player the user can play againt picks rock paper or scissors randomly every time
  • The user is allowed to select from rock paper or scissors
  • Each round the winner is determined and a log is kept of how many times the user wins vs how many times the computer wins
To check out my rock paper scissors application click the following link: Rock Paper Scissors Game
